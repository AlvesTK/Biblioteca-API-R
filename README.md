# Biblioteca API
## Python, Fastapi e Uvicorn
Uma API feita para bibliotecas simples, podendo Criar Livros e Usuarios/Leitores, retorna-los posteriormente a sua criação, além de mudar preços e adicionar descontos para os Livros da biblioteca.

Regras de Dominio 
- Livros não podem ter descontos menores que 0 ou 100.
- Leitores devem informar Email, Nome e um ID para serem cadastrados.
